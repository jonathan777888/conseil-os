# Conseil OS

Conseil OS est un système de conseillers IA qui aide l’utilisateur à réfléchir, mieux décider, s’accepter et mieux aimer les autres.

## Objectif du MVP

Créer une première version où l’utilisateur décrit une situation personnelle, puis reçoit :

1. L’avis de plusieurs conseillers
2. Une synthèse finale
3. Une validation protectrice

## Conseillers

- Conseiller sage
- Conseiller pragmatique
- Conseiller bienveillant
- Conseiller protecteur
- Conseiller principal

## Stack technique

- Python
- Flask
- Prompt Engineering
- Software Engineering
- Application Security
